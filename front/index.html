Agendamento

O sistema permite cadastrar pacientes, dentistas e consultas por meio de uma API REST integrada com banco de dados MySQL.

API REST do Projeto

Backend desenvolvido com Node.js, Express, Prisma, MySQL e JWT.

  • POST /auth/register — Cadastro de usuário
  • POST /auth/login — Login com JWT
  • GET /patients — Listar pacientes
  • POST /dentists — Cadastrar dentistas
  • POST /appointments — Criar consultas
